# OWENS Studio GUI Fixtures

This directory contains small, relocatable OWENS Studio projects for GUI and
service-layer smoke tests.

## RM2

`rm2/owens_project.yml` opens the existing `examples/RM2` WindIO and modeling
options files without copying them. The project uses relative roots so it can be
opened from any clone path, exported as a static workbench bundle, or served by
the future OWENS Studio app shell.
